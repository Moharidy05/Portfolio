Heart Disease Prediction System

Python Logistic Regression 95% Accuracy

Project Overview

This project utilizes machine learning to predict the likelihood of heart disease in patients based on medical attributes. By analyzing key health indicators such as age, blood pressure, and cholesterol levels, the model aids in the early identification of cardiovascular risks.

Technical Implementation

  • Data Preprocessing: Cleaned and normalized the `heart.csv` dataset using Pandas.
  • Model Architecture: Implemented Logistic Regression using Scikit-Learn for binary classification.
  • Validation: Utilized `train_test_split` to separate training and testing data, achieving high accuracy.
